About Melt

Melt is a minimal interpreted, object-oriented programming language implemented in C++. It is designed to be small and easy to experiment with: lexer, parser, AST, and interpreter in a single codebase.

Features

  • Variables, strings, numbers, booleans, arrays
  • Control flow: if / else, while, try / catch / throw
  • Logical operators: &&, ||, !
  • Classes and methods, this, return, import
  • Built-in HTTP server, file I/O, JSON, optional MySQL
  • Blade-like view templates for MVC-style web apps

This official website is built with Melt using the same MVC pattern: main.melt, routes, controllers, views, and static assets.
