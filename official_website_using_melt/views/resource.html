Resources

Everything you need to learn and use Melt.

Getting the interpreter

  • Build: make → produces bin/melt
  • With MySQL: make with-mysql
  • Install: make install (optional; use PREFIX=$(HOME)/.local for user install)

Documentation

  • Repo docs: docs/README.md, docs/01_GETTING_STARTED.md, docs/02_LANGUAGE_REFERENCE.md, docs/03_BUILTINS.md
  • HTML docs: docs/html/index.html (open in a browser)
